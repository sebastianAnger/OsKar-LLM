# OsKar-LLM
Oskar LLM agent

Este Repositorio es una guia basica para utilizar **Google/genai** en **Node.js** en un caso de uso sencillo y utilizar Vertex Ai en conjunto de Genkit para realizar pruebas y comportamientos de utilizar RAG.

## Herramientas
* Visual Studio Code
* Node.js: Version `v20+` superior.
* Gemini API. Puedes obtener una desde [Google AI Studio](https://aistudio.google.com), ya sea conectar una configuracion gratuita o vincular con un proyecto de Google Cloude. En mi caso utilice un nivel gratuito para hacer test de prueba y error. Despues pasar a facturacion para utilizar modelos mas recientes y probar funcionalidad y comparar resultados.
